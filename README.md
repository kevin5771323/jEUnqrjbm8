## 前言

欢迎来到Pettingpets（爱抚宠物）小程序的项目介绍。本项目旨在为广大宠物爱好者提供一个方便、快捷的在线平台，让大家能够更好地了解和关爱自己的宠物。以下是本项目的详细介绍。

## 内容介绍

Pettingpets小程序是一款基于Java语言的微信小程序，使用Spring、Springmvc、Mybatis等主流框架进行开发。通过本小程序，用户可以轻松实现宠物信息的浏览、发布和互动等功能。同时，项目前端采用Uniapp框架，结合JS、Vue和CSS3技术，为用户提供极致的交互体验。

## 技术介绍

- 语言：Java
- 使用框架：Spring、Springmvc、Mybatis，微信小程序
- 前端技术：JS、Vue、CSS3，Uniapp
- 开发工具：IDEA/Eclipse，Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
